Feature Constraints

This section lists the feature constraints and limitations on the OF-DPA reference platforms.

VXLAN

  • VxLAN Tunnelling is only supported on the BCM56850 and the BCM56960.

ACL Matches on IPv6 Addresses and MAC Address

  • Matching both IPv6 addresses and MAC address in an ACL rule is only supported on the BCM56850. On all other switches except the BCM88375, the IPv6 Flow Label can be matched instead of MAC addresses.

ACL Matches on IPv6 Addresses and L4 Ports

  • Matching both IPv6 addresses and L4 ports in an ACL rule is not supported on the BCM88375.

Termination MAC Flow Table

  • Matching on IN_PORT is not supported on the following switches:
    • BCM56150
    • BCM88375

The Dune family of switches (BCM88xxx) has the following requirements for the Termination MAC flow table:

  • The VLAN ID is required for all entries (including multicast).
  • The unicast entry for a VLAN must be created before multicast entries.
  • No more than one unique unicast MAC address per VLAN.

Using L3 ECMP Group Entries

  • For Unicast Routing flow entries with an exact match on the IP address (host entries), the Group write action may use an L3 ECMP group only on the following switches:
    • BCM56850
    • BCM56960
    • BCM88375

MPLS Support

  • MPLS is not supported on the following switches:
    • BCM56150
    • BCM56340
    • BCM88375

MPLS Flow Tables

  • On all supported platforms, MPLS flows may not specify IN_PORT as a match field. The IN_PORT field must match 'all'. This is due to the operation of the hardware.

MPLS 0 Flow Table

  • The MPLS 0 Flow table is only supported on the following switches:
    • BCM56260
    • BCM56450
    • BCM56547
    • BCM56643

MPLS ECMP Groups

  • The MPLS ECMP Group table is only supported on the following switches:
    • BCM56260
    • BCM56450
    • BCM56547
    • BCM56643

OAM

  • OAM is only supported on the following switches:
    • BCM56260
    • BCM56450
  • CCM counters are not supported.
  • CCM based loss measurement is only supported for BHH MEGs and is not supported for Ethernet MEGs.

L2 Policer Flow Table

  • The L2 Policer Flow table is only supported on the following switches:
    • BCM56260
    • BCM56450
    • BCM56547
    • BCM56643

Flow Statistics

  • Flow statistics are only supported for the following flow tables:
    • ACL Policy Flow table

VLAN flow entry for MPLS-TP VPWS Tunnel Initiation flow

  • In the following switches, to configure an MPLS VPWS Initiation flow the MPLS L2 Port Flow entry should be made before adding the VLAN Flow entry:
    • BCM56547
    • BCM56640

Size of Traffic Class

  • The size of Traffic Class is limited to 3 bits in the following switch:
    • BCM56150
  • For the rest of the switches Traffic Class is 4 bits long.
